Nodemon

So what is Nodemon? Nodemon is a very useful tool, or to be precise an npm package, that allows developers to easily restart their node applications with having to manually stop and start them over and over. The usual process is to run an application via "node app.js", then when we implement a change, we first stop the server (CTRL + c) and start it again using the same command.

Nodemon on the other hand recognizes all file changes from our directory, and as soon as a change happens, it restarts the application by stopping and starting it again itself... And for its usage, it is extremely simple to use - just instead of writing "node", we just say "nodemon" and then the .js file. When installing Nodemon, we do not install it like "express" or the "cross-env" dependency - the reason behind this is that we of course can do that, but we were suggested by our teacher to simply install "Nodemon" globally, so that we can use it for any project/ application we create and not only for a specific one.

To install "Nodemon" globally, we have to write this command in the terminal: ("npm install -g nodemon") - now it is installed globally to our system path. To get it as a dev-tool (so like "cross-env"), we run ("npm install --save-dev nodemon").
